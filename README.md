# Phonebook-backend

## About

<ul>
	<li>Додаток написаний на Node.js</li>
	<li>Сервер написано на express</li>
	<li>Збереження даних зa домопогою mongoose</li>
	<li>Дані зберігаються в MongoDB</li>
	<li>Зображення зберігаються на Cloudinary</li>
</ul>
